Перелік основних GIT команд

  • git config --global user.name blabla - user name
  • git config --global user.email blabal@bla - user email
  • git init - create an empty Git repository or reinitialize an existing one
  • git status - show the working tree status
  • git add - add file contents to the index
  • git commit - record changes to the repository
  • git log - show commit logs
  • git checkout - switch branches or restore working tree files
  • git branch - list, create, or delete branches
  • git push - update remote refs along with associated objects
  • git pull - fetch from and integrate with another repository or a local branch
  • git merge - join two or more development histories together
  • git clone - clone a repository into a new directory
  • git revert - revert some existing commits
  • git reset - reset current HEAD to the specified state
  • git rebase - reapply commits on top of another base tip
  • git cherry-pick - apply the changes introduced by some existing commits
  • git stash - stash the changes in a dirty working directory away

A . gitignore file is a text file placed in your git repository that tells git not to track certain files and folders that you don't want being uploaded to your master repository.
